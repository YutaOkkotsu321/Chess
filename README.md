ChessTech — веб-приложение для игры в шахматы против ИИ с сохранением результатов и персональной статистикой. Пользователь может выбрать цвет фигур, настроить уровень сложности движка, сыграть партию в браузере и сохранить ее в профиль для последующего просмотра.

## Что мы сделали

Мы разработали шахматный сервис, который объединяет:
- игру против Stockfish прямо в браузере;
- настройку сложности ИИ и параметров партии;
- сохранение текущего состояния игры между сессиями;
- авторизацию и личный кабинет пользователя;
- историю сыгранных партий и базовую игровую статистику.

## Для кого это

Продукт предназначен для:
- начинающих игроков, которые хотят тренироваться против ИИ;
- любителей шахмат, которым нужен быстрый и удобный онлайн-инструмент для практики;
- пользователей, которые хотят отслеживать свои результаты и прогресс в одном интерфейсе.

## Почему это ценно

ChessTech решает практическую задачу регулярной шахматной тренировки без необходимости искать соперника, устанавливать отдельный движок или вручную фиксировать результаты. Пользователь получает доступ к игре, настройкам, истории партий и персональной статистике в одном приложении. Главная ценность проекта — сделать самостоятельную шахматную практику простой, доступной и непрерывной.
